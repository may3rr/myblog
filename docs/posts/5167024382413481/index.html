《初恋》观后感

《初恋 First Love》真的好看死了，剧情虽然老套吧，但那股从胶片里溢出来的北海道冷空气，配着宇多田光的歌声，一下把我拉到旭川常盘圆环的夜色里啊

十几岁的你们相遇了，当时你的梦想是当飞行员，她想当空姐。她的母亲并不喜欢你，但你依然坚定地对她母亲保证：“我会好好守护她的！”

后来你去当了兵，你们开始了异地恋。虽然每个星期只能通三分钟电话，但你们还是好开心。只是这个时候，她却想出国读书了，却始终不知道怎么向你开口。

直到你的一个假期，你满心欢喜地跑回来找她。你留着军队里的寸头，带她去了咖啡厅，菜单上的字你看都看不懂。就在这个时候，她鼓起勇气告诉你，她想要出国读书。你完全慌了神，有点失落，也有些生气，脱口而出：“我们不是说好了永远在一起的吗？！”

你们吵了一架，你生气地扭头走了。但没走多远，你心里却不踏实，犹豫着是否该回头道歉，这时候电话响了。电话里告诉你，她出车祸了。你赶忙跑回去，面对你的是她妈妈充满愤怒的质问：“你不是说好了会好好守护她的吗？”她要求你永远不能再联系她。

后来，她失忆了，嫁了人，离了婚。而你毫不知情，只是一次次地寄信，但是寄出去的每封信都没有回应。

直到多年之后，你们又再次相遇了😥。你暗自发誓，这次一定要离得远远的，好好守护她……
